# Дизайн проекту Quiz Trainer

Ця папка містить макети та дизайн-матеріали проекту.

## Структура

- `mockups/` - макети сторінок
- `assets/` - графічні матеріали
- `wireframes/` - проволочні каркаси

## Опис дизайну

Quiz Trainer - це веб-додаток для вивчення слів через інтерактивні квізи. Дизайн орієнтований на простоту та зручність використання.
