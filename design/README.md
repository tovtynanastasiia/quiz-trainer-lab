# Дизайн проекту Quiz Trainer

Ця папка містить макети та дизайн-матеріали проекту.

## Доступні макети

- `вхід.png` — екран авторизації з полями для пошти та пароля
- `реєстрація.png` — сторінка створення нового облікового запису
- `головний екран.png` — стартовий екран із загальним прогресом і швидкими діями
- `квізи режим навчання.png` — сценарій практичного проходження квізу
- `режим точність.png` — інтерфейс перевірки відповідей на точність
- `керування наборами.png` — список наборів із можливістю додавання та видалення
- `редагування набору.png` — екран редагування слів у вибраному наборі

## Опис дизайну

Quiz Trainer — це веб-додаток для вивчення слів через інтерактивні квізи. Дизайн орієнтований на простоту, зрозумілі навчальні сценарії та підтримку різних режимів тренування.

> Щоб додати нові макети, додайте зображення до цієї директорії та доповніть список коротким описом.
