# Quiz Trainer – Lab 3 foundation

Ця гілка містить базову структуру додатка на **Vite + React + TypeScript** для подальших лабораторних робіт. На даному етапі реалізовано каркас інтерфейсу без бізнес-логіки: мінімум три сторінки та модулі аутентифікації.

## Стек

- Vite 7
- React 19 + TypeScript
- ESLint 9 (конфіг від Vite)

## Запуск

```bash
npm install
npm run dev
```

Готовий сервер буде доступний за адресою [http://localhost:5173](http://localhost:5173).

## Структура директорій

```
src/
├── components/         # Базові UI-компоненти без бізнес-логіки
├── layouts/            # Макети сторінок
├── pages/              # Представлення сторінок (Home, Word Sets, Learning Modes, Auth)
├── styles/             # Глобальні стилі й токени
├── types/              # Спільні типи
└── main.tsx            # Точка входу
```

## Реалізовані екрани

- `Home` — огляд прогресу та запланованих сесій
- `Learning modes` — опис доступних режимів тренувань
- `Word sets` — управління наборами слів
- `Sign in` / `Register` — форми авторизації (без інтеграції)

## Lab 5 — маршрутизація

- Підключено `react-router-dom` та налаштовано маршрути для всіх екранів.
- Додано сторінку `404` з можливістю повернутися назад або на головну.
- Реалізовано динамічний та вкладений маршрут `/sets/:setId` з отриманням параметра URL.
- Створено захищений маршрут `/account` із симуляцією аутентифікації (швидке перемикання статусу).
- Продемонстровано програмний редірект та навігацію по історії (`navigate(-1)`).
